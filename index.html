Flexbox Example

To initiate a flexbox layout, there must be a container/wrapper with the value of display:flex that will permit children to employ flex values. All direct children of the flex container automatically become flex items.

By default, flexbox distributes width values on adjacent columns depending on which one has the most content, with adjacent columns distributing the remainder. To resolve this, set the flex container's class to .container > * {width: 100%} if you want all columns to have the same width.

The baseline value to evenly distribute the horizontal space of elements is flex 1. Flex values specify how much the item will grow relative to the rest of the flexible items. Here, this main column has .main-column {flex: 3;} whereas the asides each have flex: 1

The flex container can be given a flex-directon property to set the display direction of its child elements to either column or row. Element order can also be reversed by column-reverse or row-reverse

For multi-element layouts, such as this main content panel bordered by asides, set the element order properties to the desired numeric order (1,2,3). The flexbox container by default has a flex-direction of row, which forces its child elements into a columned layout within a row. Here we override this by giving the container of these two child elements a column quality with flex-direction: column;

B

Flex-direction can be used in media queries to change from a row to column at a specific breakpoint. This enables child elements to take up rows of horizontal space, sort of like block elements.

Media queries can also be used to change the width of the flex property on different screen sizes. To do so requires adding flex-wrap: wrap; to the flex container.

Whereas Flexbox focuses primarily on rows, CSS Grid properties apply both to rows and columns. However, Flexbox easily handles a combination of rows and columns as necessary.
To set a grid layout, a container must first have a display property set to either display: grid; or display: inline-grid;. All direct children of a grid container will automatically become grid items.
Vertical lines of grid items are set with grid-column:, and horizontal lines are set by grid-row:.

A grid container has one grid item for each column in each row by default, but grid items can be styled so that they span multiple columns and/or rows by setting its grid-column or grid-row to something like 1 / span 3 or 2 / 4, resulting in the element expanding to span 3 regular columns/rows.
