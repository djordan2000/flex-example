Flexbox Example

To initiate a flexbox layout, there must be a container/wrapper with the value of display:flex that will permit children to employ flex values. All direct children of the flex container automatically become flex items.
The baseline value to evenly distribute the horizontal space taken up by elements is flex 1. Flex values specify how much the item will grow relative to the rest of the flexible items. Here, this main column has .main-column {flex: 3;} whereas the asides each have flex: 1
The flex container can be given a flex-directon property to set the display direction of its child elements to either column or row. Element order can also be reversed by column-reverse or row-reverse

For multi-element layouts, such as this main content panel bordered by asides, set the element order properties to the desired numeric order (1,2,3). Centering Elements

Flexbox centers elements very simply.
Just give the container for the centered element { display: flex;}
All that's necessary to center an element both vertically and horizontally in a flex container is to give it a margin of auto .center-me {margin: auto;}

Grid's CSS properties apply to both rows and columns, whereas Flexbox focuses primarily on rows, although they can be combined as necessary according to different element requirements.
To set a grid layout, a container must first have a display property set to either display: grid; or display: inline-grid;. All direct children of a grid container will automatically become grid items.
Vertical lines of grid items are set with grid-column:, and horizontal lines are set by grid-row:.

A grid container has one grid item for each column in each row by default, but grid items can be styled so that they span multiple columns and/or rows by setting its grid-column or grid-row to something like 1 / span 3 or 2 / 4, resulting in the element expanding to span 3 regular columns/rows.
